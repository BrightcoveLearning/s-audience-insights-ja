Net Customer Change

See how your customer count changes over time. This graph includes projections and trends and milestones.

Actions

You can select any of the following actions to change the data presented:

  • Click to download customer change data as a CSV file or a PNG image.
  • Select a date range for your data. Note that the date range should be 5 days or longer.
  • Select the x-axis granularity to be Day, Week or Month.
  • Check to automatically change the x-axis dimension.

Data

The following data is displayed in the graph:

Projections are a linear regression analysis based on customer counts over the last 12 weeks. Up to 3 months of predictive data is available.
  • Total Customers - Total number of customers
  • Total Customers (projected) - Total projected number of customers
  • Net Change - Number of customers gained minus number of customers lost
  • Net Change - Number of projected net customers

Trends

See the four week running average number of customers as well as the future trends for customer count.

Data

The following data is displayed above:

  • Projections (total customer count) - Total projected number of customers for the future
  • Four week running average (net customer change) - Net customer change in customer count

Milestones

Create a milestone to track how it effects your customer count.

Actions

You can select any of the following actions:

  • Click View to filter data by:
    • Operations
    • Product
    • Business
    • Content
    • Marketing

  • >Click to create a new milestone. Fill in the form fields and select ADD when complete.


  • Hover over the graph to see the data and associated milestone.
